# 🎬 AI Tools Faceless Channel — Script Generator

A Python script that automatically generates a professionally formatted PDF containing **4 weeks of ready-to-use YouTube/TikTok/Reels scripts** for an AI tools faceless channel.

## 📋 What's Inside

- **12 fully scripted videos** across 4 weeks of content
- **Voiceover scripts** optimized for AI voice generators (ElevenLabs, etc.)
- **Seedance 2 video prompts** using the proven formula:
  `[Subject] + [Action] + [Camera Move] + [Style] + [Lighting] + [Audio] + [Duration]`
- Covers **YouTube long-form**, **TikTok**, and **Instagram Reels (9:16)**

## 📅 Content Schedule

| Week | Theme | Videos |
|------|-------|--------|
| Week 1 | Launch & Establish | Videos 1–2 |
| Week 2 | Build Audience | Videos 3–5 |
| Week 3 | Monetization Focus | Videos 6–8 |
| Week 4 | Scale & Repeat | Videos 9–12 |

## 🚀 Getting Started

### Prerequisites

- Python 3.7+
- `reportlab` library

### Installation

```bash
pip install -r requirements.txt
```

### Usage

```bash
python generate_pdf.py
```

This will generate `AI_Tools_Faceless_Channel_Scripts.pdf` in the same directory.
